# Flower Shop Telegram Bot

## Описание
Телеграм бот для продажи букетов из атласных роз. Поддерживает каталог, корзину, оформление заказа и админ-панель.

## Установка и запуск

1. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
   python -m venv venv
   # Windows
   venv\Scripts\activate
   # Linux/Mac
   source venv/bin/activate
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройка окружения:**
   Файл `.env` уже создан. Убедитесь, что `BOT_TOKEN` и `ADMIN_IDS` указаны верно.

4. **Запуск бота:**
   ```bash
   python bot.py
   ```

## Структура проекта
- `bot.py`: Главный файл запуска.
- `config.py`: Конфигурация.
- `data/`: YAML файлы базы данных.
- `database/`: Модуль работы с БД.
- `handlers/`: Обработчики сообщений (клиент, админ).
- `keyboards/`: Клавиатуры.
- `images/`: Фотографии букетов.

## Функционал
- **Клиент**: Просмотр каталога, избранное, корзина, конструктор заказа.
- **Админ**: Статистика, управление заказами, добавление/удаление букетов.
